# Source Map

This top-level `src/` folder is included so the repository has a clear source entry point for reviewers.

## Actual source locations

- Frontend source: `frontend/src/`
- Backend source: `backend/app.py`, `backend/import_dataset.py`
- Frontend public assets: `frontend/public/`

## Main entry points

- React app root: `frontend/src/index.js`
- Main UI component: `frontend/src/App.js`
- Graph rendering: `frontend/src/components/GraphCanvas.js`
- Flask API: `backend/app.py`
