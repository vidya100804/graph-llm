# ============================================
#  HOW TO RUN — Dodge AI O2C Graph Explorer
# ============================================

## ⚡ FASTEST — Open Directly in Browser (No Setup)
1. Just open `index.html` in Chrome or Firefox
2. The full graph loads immediately — no server needed!
3. Get an OpenRouter key at https://openrouter.ai/keys or a Groq key at https://console.groq.com
4. Choose the provider in the yellow bar at the top of the app and paste the key
5. Start asking questions!

---

## 🖥️ Run with Backend (Full Flask Server)

### Prerequisites
- Python 3.8+
- Node.js 16+ (only if running React version)

### Step 1 — Get an LLM API key
Option A: OpenRouter → https://openrouter.ai/keys → Create key
Option B: Groq → https://console.groq.com → API Keys → Create key

### Step 2 — Install & Run Backend
```bash
cd backend
pip install -r requirements.txt
set OPENROUTER_API_KEY=your_openrouter_key_here
set OPENROUTER_MODEL=openai/gpt-4o-mini
python app.py

# Or use Groq instead
set GROQ_API_KEY=your_groq_key_here
python app.py

# Mac/Linux
export OPENROUTER_API_KEY=your_openrouter_key_here
export OPENROUTER_MODEL=openai/gpt-4o-mini
python app.py
```
Backend runs at: http://localhost:5000

### Step 3 — Open the App
Option A (easiest): Just open index.html in browser
Option B (React dev): 
  cd frontend
  npm install
  npm start
  Open http://localhost:3000
